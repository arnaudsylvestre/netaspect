Prepare your project

For Visual Studio 2008

Download the latest version of NetAspect Here and unzip it in the directory you want. We well call this directory : NetAspectDirectory

For Visual Studio 2010/2012

On the project you want to weave, add the nuget package NetAspect and that's all !
