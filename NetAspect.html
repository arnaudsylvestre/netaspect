NetAspect

An open source AOP framework for .Net and .Net Compact Framework

NetAspect is simple

Just define your aspect and follow the documentation

NetAspect is open source

It is under the apache 2 license

NetAspect is light

It does not need references at execution time

NetAspect is Compact Framework compliant

It can be used for Smart Device applications
